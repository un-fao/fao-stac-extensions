# OGC Metadata SWG Coordination Note — Draft

Placeholder for the coordination note shared with the OGC Metadata
Standards Working Group (ISO TC 211 liaison) covering the `iso-to-stac`
extension.

There is no existing STAC ↔ ISO 19115 mapping extension; this proposal
fills that gap and benefits from upstream coordination with both the
OGC and ISO TC 211 communities.

To be filled in alongside the v0.2.0 authoring iteration, once the
mapping table at `extensions/iso-to-stac/mapping/iso19115-2-to-stac.md`
is stable.
