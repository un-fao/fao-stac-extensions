# Code of Conduct

This project adopts in full the
[STAC Specification Code of Conduct](https://github.com/radiantearth/stac-spec/blob/master/CODE_OF_CONDUCT.md).

The STAC Code of Conduct is itself adapted from the
[Contributor Covenant](https://www.contributor-covenant.org/), version
1.4. Please read the upstream document for the complete text covering:

- Expected behaviour
- Unacceptable behaviour
- Reporting procedures
- Enforcement

## Reporting violations

Reports of unacceptable behaviour in this repository (issues, PRs,
discussions) should be sent to `copyright@fao.org`. Reports involving
the wider STAC community should additionally follow the upstream
reporting procedure linked above.
